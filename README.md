# skill-from-conversation

An [opencode](https://opencode.ai) skill that gathers learnings from a conversation and converts them into a reusable skill file.

## Installation

Copy the `SKILL.md` file into your opencode skills directory:

```bash
mkdir -p ~/.config/opencode/skills/skill-from-conversation
cp SKILL.md ~/.config/opencode/skills/skill-from-conversation/
```

## Usage

Once installed, the skill is available in opencode. Trigger it when you want to save a multi-step workflow from the current conversation as a reusable skill.
